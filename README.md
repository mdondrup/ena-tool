# ena-tool
A simple command-line tool to create and manage ENA projects.

A Perl script that allows to create submission projects and umbrella projects in ENA.

## Installation

Don't need to install anything, but just in case.

    perl Makefile.PL
    make install

  ### Dependencies  
  
- Perl >5.20
- 'Getopt::Long';
- 'LWP::UserAgent';
- 'HTTP::Request::Common';
- 'MIME::Base64';
- 'Pod::Usage';
- 'POSIX';
- 'XML::Simple';
- 'File::Temp';
- 'Data::Dumper';
